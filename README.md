# MNIST KNN and K-Means Project

This project explores handwritten digit classification using machine learning techniques on the MNIST dataset.

## Technologies Used
- Python
- NumPy
- Jupyter Notebook

## Project Overview
- Applied K-Means clustering to group handwritten digits
- Used k-Nearest Neighbors (k-NN) for classification
- Compared clustering and classification performance
- Explored ways to reduce dataset size while maintaining accuracy

## Skills Demonstrated
- Machine Learning Basics
- Data Analysis
- Numerical Computation
- Classification Algorithms
